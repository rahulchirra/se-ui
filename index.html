UNIT-I: BAYES' THEOREM (5 Marks)

1 BAYES' THEOREM

$$P(A|B) = \frac{P(A) P(B|A)}{P(A) P(B|A) + P(A') P(B|A')}$$

üìå PROBLEM - Machines (Standard)

M1 (60%, 3% def), M2 (40%, 5% def). Item is defective. Prob it's from M1?

Step 1: Total Probability P(D)

$$P(D) = 0.6(0.03) + 0.4(0.05) = 0.018 + 0.020 = 0.038$$

Step 2: Bayes Formula

$$P(M_1|D) = \frac{0.6 \times 0.03}{0.038} = \frac{0.018}{0.038} = 0.474$$

üìå PROBLEM - Urns

Urn I (3W, 4B), Urn II (5W, 3B). Transfer 1, then draw from II (White). Prob transferred was White?

P(W1 transf) = 3/7, P(B1 transf) = 4/7.

If W1 transf, II has 6W, 3B -> P(W2|W1) = 6/9.

If B1 transf, II has 5W, 4B -> P(W2|B1) = 5/9.

$$P(W_2) = \frac{3}{7}\frac{6}{9} + \frac{4}{7}\frac{5}{9} = \frac{18+20}{63} = \frac{38}{63}$$ $$P(W_1|W_2) = \frac{18/63}{38/63} = \frac{18}{38} = \frac{9}{19}$$

UNIT-II: BINOMIAL & POISSON (15 Marks)

2 BINOMIAL DISTRIBUTION

$$P(X = r) = \binom{n}{r} p^r q^{n-r}$$

Mean = np | Variance = npq

Type 1: Coins (Basic)

Toss 6 coins. $n=6, p=0.5, q=0.5$.

(a) Exactly 4 heads:

$$P(4) = \binom{6}{4}(0.5)^6 = 15 \times \frac{1}{64} = \frac{15}{64}$$

(b) At least 4 ($X \ge 4$):

$$P(4)+P(5)+P(6) = \frac{15+6+1}{64} = \frac{22}{64}$$

Type 2: Find n, p

Mean=6, Var=4.

1. $q = \frac{Var}{Mean} = \frac{4}{6} = \frac{2}{3}$

2. $p = 1 - q = \frac{1}{3}$

3. $n \times \frac{1}{3} = 6 \implies n = 18$

P(X=0): $\binom{18}{0}(\frac{1}{3})^0(\frac{2}{3})^{18} = (0.667)^{18}$

Type 3: Defective Items

5% defective. $n=20, p=0.05$. Find $P(X \ge 1)$.

Use Shortcut: $1 - P(X=0)$

$$P(X \ge 1) = 1 - (0.95)^{20} = 1 - 0.358 = 0.642$$

3 POISSON DISTRIBUTION

$$P(X = r) = \frac{e^{-\lambda} \lambda^r}{r!}$$

Mean = Variance = $\lambda$

üìå PROBLEM - Standard Example

Question: $\lambda = 2$. Find $P(X=0)$ and $P(X \ge 1)$. ($e^{-2} = 0.1353$)

(a) P(X=0): $$P(0) = \frac{e^{-2} 2^0}{0!} = 0.1353$$
(b) P(X \ge 1): $$1 - P(0) = 1 - 0.1353 = 0.8647$$

üìå PROBLEM - Accidents (Mean Given)

Question: Avg 3 accidents/week. Find $P(X \ge 2)$.

Given $\lambda = 3$. Formula: $1 - [P(0) + P(1)]$

$$P(0) = e^{-3} = 0.0498$$ $$P(1) = \frac{e^{-3}3^1}{1!} = 0.1494$$ $$P(X \ge 2) = 1 - (0.0498 + 0.1494) = 0.8008$$

UNIT-III: NORMAL DISTRIBUTION (10 Marks)

4 NORMAL DISTRIBUTION

$$Z = \frac{X - \mu}{\sigma}$$

üìå PROBLEM - Type 1: Probabilities

Question: Mean=165, SD=6. Find P(160 < X < 170).

1. Convert to Z:

$Z_1 = \frac{160-165}{6} = -0.83$

$Z_2 = \frac{170-165}{6} = 0.83$

2. Use Symmetry:

$$P(-0.83 < Z < 0.83) = 2P(Z < 0.83) - 1$$ $$= 2(0.7967) - 1 = 0.593$$

üìå PROBLEM - Type 2: Inverse (Top 5%)

Question: Mean=70, SD=12. Find marks for top 5%.

Top 5% means $P(Z < z) = 0.95$. Table value $z = 1.645$.

$$1.645 = \frac{X - 70}{12}$$ $$X = 70 + 19.74 = 89.74$$

5 GENERAL p.d.f

Given $f(x) = kx^2$ for $0 \le x \le 2$. Find k.

$$\int_0^2 kx^2 dx = 1 \implies k[\frac{x^3}{3}]_0^2 = 1$$ $$k(\frac{8}{3}) = 1 \implies k = \frac{3}{8}$$

UNIT-V: HYPOTHESIS TESTING (20 Marks)

9 Z-TEST FOR SINGLE MEAN

Formula:

$$Z = \frac{\bar{x} - \mu}{\frac{\sigma}{\sqrt{n}}}$$

Use when: $n \ge 30$, testing one population mean.

Standard Steps:

  1. Write $H_0$ and $H_1$
  2. Note given data ($n, \bar{x}, \sigma, \mu, \alpha$)
  3. Calculate $Z$
  4. Find critical value from table
  5. Compare and decide
  6. Write conclusion

üìå PROBLEM - Type 1: Two-tailed test

Question: A company claims average bulb life is 1200 hours. Sample of 50 bulbs: mean = 1180 hours, S.D = 100 hours. Test at 5% level.

Step 1: Hypotheses

$H_0: \mu = 1200$ (claim is true)

$H_1: \mu \ne 1200$ (claim is false) - Two-tailed

Step 2: Given

$n = 50, \bar{x} = 1180, \sigma = 100, \mu = 1200, \alpha = 0.05$

Step 3: Calculate Z

$$Z = \frac{1180 - 1200}{\frac{100}{\sqrt{50}}} = \frac{-20}{\frac{100}{7.071}} = \frac{-20}{14.14} = -1.414$$

$|Z| = 1.414$

Step 4 & 5: Decision

Critical Value (5% two-tailed): $Z_{0.05} = 1.96$

$|Z| = 1.414 < 1.96$

Accept $H_0$

Conclusion: At 5% level, company's claim is VALID.

üìå PROBLEM - Type 2: One-tailed test

Question: A principal claims students score at least 70 on average. Sample of 100 students: mean = 68, S.D = 15. Test at 5% level.

$H_0$: $\mu = 70$

$H_1$: $\mu < 70$ (Left-tailed because "at least" means $\ge$)

$$Z = \frac{68 - 70}{\frac{15}{\sqrt{100}}} = \frac{-2}{1.5} = -1.333$$

Critical value: $Z_{0.05} = -1.645$ (Left-tailed)

Decision: $|Z| = 1.333 < 1.645 \rightarrow$ Accept $H_0$

Conclusion: Claim is valid.

Key Points to Remember:

  • Two-tailed: $H_1$ uses $\ne$ (claim "equal to")
  • Right-tailed: $H_1$ uses $>$ (claim "at most")
  • Left-tailed: $H_1$ uses $<$ (claim "at least")
  • Critical Values (5%): Two-tailed = 1.96, One-tailed = 1.645

10 Z-TEST DIFFERENCE OF MEANS

Formula:

$$Z = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}}$$

Use when: $n_1, n_2 \ge 30$.

üìå PROBLEM - Standard Type

Question: Sample 1: $n_1=100, \bar{x}_1=65, \sigma_1=8$. Sample 2: $n_2=80, \bar{x}_2=60, \sigma_2=7$. Test difference at 5%.

$H_0$: $\mu_1 = \mu_2$

$H_1$: $\mu_1 \ne \mu_2$

Calculation:

$$Z = \frac{65 - 60}{\sqrt{\frac{64}{100} + \frac{49}{80}}} = \frac{5}{\sqrt{0.64 + 0.6125}} = \frac{5}{1.119} = 4.468$$

$|Z| = 4.468 > 1.96$

Reject $H_0$

Conclusion: Means differ significantly.

üìå PROBLEM - Word Problem (Boys vs Girls)

Question: Boys ($n=120$): mean=72, SD=12. Girls ($n=100$): mean=70, SD=10. Test at 1% level.

$$Z = \frac{72 - 70}{\sqrt{\frac{144}{120} + \frac{100}{100}}} = \frac{2}{\sqrt{1.2 + 1}} = \frac{2}{1.483} = 1.349$$

Critical value (1%): 2.58

Decision: $1.349 < 2.58 \rightarrow$ Accept $H_0$

No significant difference at 1% level.

üìå PROBLEM - Complete Solution

Question: $n_1=8, \bar{x}_1=12, s_1=2$ | $n_2=10, \bar{x}_2=10, s_2=1.5$. Test at 5%.

Step 1: Pooled Variance ($s^2$)

$$s^2 = \frac{7(4) + 9(2.25)}{16} = \frac{28 + 20.25}{16} = 3.016$$ $$s = \sqrt{3.016} = 1.737$$

Step 2: Calculate t

$$t = \frac{12 - 10}{1.737\sqrt{\frac{1}{8} + \frac{1}{10}}} = \frac{2}{1.737(0.474)} = \frac{2}{0.823} = 2.431$$

Step 3: Decision

d.f. = $8+10-2 = 16$. Table value $t_{0.05}(16) = 2.120$.

$|t| = 2.431 > 2.120$

Reject $H_0$

MUST LEARN DEFINITIONS (2 Marks)

Unit I

  • Bayes Theorem: Formula for conditional prob $P(A|B)$.
  • Independent Events: $P(A \cap B) = P(A)P(B)$.
  • Conditional Prob: $P(A|B) = P(A \cap B) / P(B)$.

Unit II

  • Binomial Mean/Var: Mean=np, Var=npq.
  • Poisson Mean/Var: Mean = Variance = $\lambda$.
  • Poisson Approximation: Use when n is large, p is small (np = $\lambda$).

Unit III

  • Normal Properties: Symmetric, Mean=Median=Mode, Area=1.
  • Area limits: 95% data within $\mu \pm 1.96\sigma$.
  • PDF Property: $\int_{-\infty}^{\infty} f(x)dx = 1$.

Unit IV

  • Correlation vs Regression: Correlation measures strength; Regression predicts.
  • Correlation Properties: $-1 \le r \le 1$, independent of origin/scale.

Unit V

  • Null Hypothesis ($H_0$): Hypothesis of "no difference".
  • Alt Hypothesis ($H_1$): Hypothesis that contradicts $H_0$.
  • Type I Error ($\alpha$): Reject $H_0$ when true.
  • Type II Error ($\beta$): Accept $H_0$ when false.
  • Level of Significance: Probability of Type I error (e.g., 5%).
